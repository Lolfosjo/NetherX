A PocketMine-MP API 5 plugin that brings the Minecraft 1.16 Nether to PocketMine-MP!

## Features

- **All Nether Biomes** (Warped Forest, Crimson Forest, Basalt Delta, Soulsand Valley, Nether Wastes)
